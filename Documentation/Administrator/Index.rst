Administrator Manual
====================

Target group: **Administrators**

.. _admin-installation:

Installation
------------

To install the extension, perform the following steps:

#. Go to the Extension Manager
#. Install the extension

The default configuration will show a dark bar without a link to the cookie policy at the bottom of the page.

After installing the extension the TypoScript configuration will be included on all pages in your installation.
If you want to include the files per site or as a dependency of a site package you can disable this feature
by going into the Extension Manager and checking the option to disable autoloading TypoScript constants and setup.
